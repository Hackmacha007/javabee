Sum of Two Digits

This Java code calculates the sum of two digits:

      public class SumOfTwoDigits {
        public static void main(String[] args) {
          int num1 = 5;
          int num2 = 7;
          int sum = num1 + num2;
          System.out.println("The sum of " + num1 + " and " + num2 + " is " + sum);
        }
      }
